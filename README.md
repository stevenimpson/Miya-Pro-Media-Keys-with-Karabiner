# Miya-Pro-Media-Keys-with-Karabiner
Access media keys with Fn+numbers without switching to function layer on Miya Pro Mac keyboard

Copy this file into 
    ~/.config/karabiner/assets/complex_modifications


| Key Combo  | Command   |
|---|---|
| Fn + 7  | Rewind  |
| Fn + 8  | Play/Pause  |
| Fn + 9  | FastForward  |
